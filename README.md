# 📊 Sales Data Analysis Dashboard

**Tool:** Microsoft Power BI
**Author:** Mariya Tanveer
**Location:** Glen Ellyn, IL

## 📌 Project Overview
An interactive Power BI dashboard analyzing sales performance,
revenue trends, and regional breakdowns to support data-driven decisions.

## 📊 Key Insights
- Total revenue and profit KPIs at a glance
- Month-over-month and year-over-year sales trends
- Regional and product category performance breakdown
- Dynamic filters for time period and category
